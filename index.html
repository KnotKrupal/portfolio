Builds

Selected projects & experiments

Python • Internal Tooling

SmartImprint-style Artwork Manager

A Python GUI that searches, validates and routes artwork (EPS/CDR) from complex network folders into the correct print queues for front, reverse, pen and customer-specific jobs.

  • Python
  • Tkinter GUI
  • Multithreading
  • File automation

Features include fallback searches, per-job status tracking, configurable paths, and clear operator feedback – turning “hunt through 10 folders” into “click once and print confidently.”

Laser • Mechanics • 3D Print

Automatic Pen Laser Engraving Concept

A single-pen automated engraving pipeline for an OMTech MOPA 20W laser, using a rotary chuck, guides and future Arduino/ESP32 control for synchronized load–engrave–unload cycles.

  • Mechanism design
  • 3D-printed fixtures
  • Automation concept

The design targets ~4 pens/min, focuses on repeatable alignment, and is built to evolve toward a fully sensor-backed, safety-aware automation cell.

Robotics • R&D

Robot Arm & Auto-Feeder Research

Research into lightweight robotic arms and feeders that can load printers or lasers, using off-the-shelf stepper solutions and 3D-printed parts to keep costs down.

  • Open-source robotics
  • Motion planning
  • Low-cost automation

Explored servo/stepper driver options, end-effector ideas and how a cell like this could talk to print/engrave jobs for synchronized production.
